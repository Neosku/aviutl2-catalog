# AviUtl2 カタログ

AviUtl2 のプラグインやツールを「探す・知る・入れる・更新する」を一気通貫で行える、デスクトップ向けカタログアプリです。  
Tauri 2 + React 19 (Vite 7) を利用して実装しています。
現在ベータ版です。バグなどがあれば本ソフト内のフィードバッグからお知らせください。

---

## 機能概要
- 🔍 パッケージの検索  
- 📦 パッケージのインストール / 更新 / 削除  
- 🔔 AviUtl2 の画面上で、インストール済みプラグインに更新がある場合に通知
- 🧩 インストール済みパッケージの自動検出（XXH3-128 によるハッシュ）  
- 📨 パッケージの投稿（検査後に反映）

## 画面構成
- **ホーム**: パッケージの一覧表示、検索、フィルタを用いたソート
- **パッケージ詳細**: パッケージの詳細表示ページ、パッケージのイメージ画像、md形式の説明  
- **フィードバック**: フィードバック送信（パッケージ登録 / 不具合報告 / 意見・問い合わせ）

## 開発環境・ビルド要件
- OS: **Windows 11 x64**
- Node.js: **v18 以上**
- Rust & Tauri 2 開発環境

## 保存先（AppConfig 配下）

Tauri のアプリ設定ディレクトリに以下のデータを保存します。

* `settings.json` : AviUtl2 のパスやテーマの設定を保存
* `catalog/index.json` : 取得したカタログデータのキャッシュ
* `installed.json` : インストール済み ID 一覧
* `hash-cache.json` : ハッシュ計算のキャッシュ
* `logs/app.log` : アプリログ
* `installer-tmp/<id-version>/` : インストーラーの一時保存フォルダ

## カタログデータ

* パッケージ登録は、アプリ内の **フィードバック** からお願いします。（作者以外の方も大歓迎です）
* カタログデータは`aviutl2-catalog-data`のindex.jsonに記録しています。30分ごとに更新を自動確認します(対応しているもののみ)
（[パッケージ一覧](https://github.com/Neosku/aviutl2-catalog-data/blob/main/パッケージ.md)）
